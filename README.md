## 🧪 Recommendation systems & ML projects

**[mind-graph-rec](https://github.com/DanielaBalaniuc/mind-graph-rec)**
A heterogeneous graph neural network recommender on Microsoft's MIND news
dataset (PyTorch Geometric, GraphSAGE). Models users, articles, and
Wikidata knowledge-graph entities jointly; trained with BPR loss on real
impression-level negatives. Diagnosed a cold-start evaluation gap by
directly measuring train/dev user overlap (11.9%) rather than taking the
offline metrics at face value — full write-up in the repo.

**[newslens-ai-v2](https://github.com/DanielaBalaniuc/newslens-ai-v2)**
A hybrid news recommender (content-based + collaborative filtering) on the
MIND dataset, using sentence embeddings and FAISS for retrieval, evaluated
via Precision@K / Recall@K / MAP / NDCG. Work in progress.

**[Building a RAG Pipeline using LangChain and FAISS](https://github.com/DanielaBalaniuc/Building-a-RAG-Pipeline-using-LangChain-and-FAISS)**
A retrieval-augmented generation pipeline over PDF documents, using
LangChain and FAISS for vector-based semantic retrieval.
